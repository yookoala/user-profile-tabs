Regroup DOM element in profile form into tabs for better UX.


== Description ==

If your User Profile page is way too stuffed with fields (e.g. from Paid Membership Pro), this plugin will
inject CSS and Javascripts to re-organize sections in "Account Management" sections into tabs. All fields
under H3 will be re-organized in tabs.
